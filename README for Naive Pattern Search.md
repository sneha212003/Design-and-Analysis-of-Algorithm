# Naive Pattern Searching Algorithm in Python

This project implements the Naive Pattern Searching Algorithm using Python.

## Features
- Searches for a pattern inside a text
- Returns starting index
- Measures execution time

## Example

Input:
```python
Enter a text : banana
Enter a pattern : ana
```

Output:
```python
Pattern found at index : 1
Execution Time : 0.00001
```

## How to Run

```bash
python pattern_search.py
```
