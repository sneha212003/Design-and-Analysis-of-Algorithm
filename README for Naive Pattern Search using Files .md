# Naive Pattern Searching Algorithm using File Handling

This project implements the Naive Pattern Searching Algorithm in Python using text file input.

## Features
- Reads text from `input1.txt`
- Reads pattern from `pattern.txt`
- Searches for the first occurrence of the pattern
- Displays:
  - Match index
  - Text length
  - Pattern length
  - Execution time

## Technologies Used
- Python
- File Handling
- Time Module

## Example

### input1.txt
banana

### pattern.txt
ana

### Output
Pattern found at index : 1
Text Length : 6
Pattern Length : 3
Execution Time : 0.00001

## How to Run

```bash
python pattern_search.py
```
